# DH-Blog

## 技术栈

### 后端部分

- JDK21 + SpringBoot3
- MySQL + MybatisPlus
- Redis

### 前端部分

- TS + Vue3 + ElementPlus
- Axios
- [md-editor-v3](https://github.com/imzbf/md-editor-v3) `作为 markdown 编辑器`
